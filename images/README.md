# Images Folder

## Upload Your Logo Here

### Required Image:
- **Filename:** `muse-logo.png`
- **Size:** 500x500px (square)
- **Format:** PNG with transparent background
- **Colors:** Use green (#00D170) accent

### Quick Logo Creation:

1. **Option 1: Canva (Free)**
   - Go to canva.com
   - Create 500x500px design
   - Add "MUSE" text in white
   - Add music note icon
   - Set background to dark (#0A0A0F)
   - Add green accent (#00D170)
   - Download as PNG

2. **Option 2: Figma (Free)**
   - Create new file
   - Add 500x500 frame
   - Design your logo
   - Export as PNG

3. **Option 3: Use AI**
   - Use DALL-E or Midjourney
   - Prompt: "Modern logo for MUSE music app, green neon color, dark background, music note, minimalist"

### After Creating Logo:

1. Name it exactly: `muse-logo.png`
2. Upload to this folder on Porkbun
3. URL will be: `https://musemint.io/images/muse-logo.png`
4. This URL is already referenced in metadata.json!

### Optional Additional Images:

- `favicon.ico` - Browser tab icon (32x32px)
- `og-image.png` - Social media share image (1200x630px)
- `hero-bg.jpg` - Hero section background (1920x1080px)

---

**For now, the site uses an SVG placeholder icon, so the logo is optional but recommended for branding!**
